# BRIDGE
Paper： Relation-aware Adaptive Multimodal Fusion and Bidirectional Ranking Consistency for Multi-Modal Knowledge Graph Completion
BRIDGE is a framework for Multi-modal Knowledge Graph Completion (MMKGC).

## Environment

- Python 3.10

It is recommended to create a new Python 3.10 environment before running the project.

```bash
conda create -n BRIDGE python=3.10
conda activate BRIDGE
```

## Installation

Install all required dependencies:

```bash
pip install -r requirements.txt
```

## Running

After installing the dependencies, run:

```bash
python BRIDGE.py
```
